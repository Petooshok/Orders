## Описание проекта

Этот репозиторий является частью курса "Программирование корпоративных систем" и посвящён третьему этапу обучения, ориентированному на создание корпоративных приложений. Основной целью проекта является разработка системы управления товарной корзиной в мобильном приложении с функционалом для добавления, редактирования и удаления позиций. В рамках проекта были внедрены новые технологии и инструменты, такие как ***Firebase***, который теперь используется для управления данными, создания и оформления заказов.

## Цель проекта

Целью проекта является изучение основ разработки корпоративных систем и закрепление навыков Full-stack разработки. Проект включает работу с пользовательским интерфейсом и серверной частью для обработки данных, а также интеграцию с ***Firebase*** для управления базой данных, аутентификации пользователей и создания заказов.

## В рамках дисциплины

На данном этапе курса студенты изучают:

- **Мобильная разработка под Android и iOS**: Применение Flutter и Dart для создания кроссплатформенных приложений.
- **Взаимодействие с данными**: Работа с REST API для обработки данных на сервере и клиенте, а также использование ***Firebase*** для управления базой данных, аутентификации и создания заказов.
- **Пользователи могут взаимодействовать** только с продавцом, а продавец, в свою очередь, может общаться только с теми пользователями, которые ему написали. Это единственный функционал, доступный на странице продавца.

## Новые возможности

В рамках текущего этапа проекта были реализованы следующие улучшения:
- **Вход и регистрация пользователей** с использованием ***Firebase Authentication***.
- **Пользовательский чат** для общения с продавцом.
- **Создание и оформление заказов** с хранением данных в ***Firebase Firestore***.

Все эти функции теперь работают с использованием ***Firebase***, что позволяет упростить управление данными и повысить эффективность разработки.

## Заключение

Проект предоставляет возможность более детально изучить разработку корпоративных приложений, а также закрепить практические навыки работы с кроссплатформенными технологиями и управлением пользовательским интерфейсом. Внедрение ***Firebase*** позволило упростить управление данными, аутентификацию пользователей и создание заказов, что значительно повысило эффективность разработки и удобство использования приложения.
